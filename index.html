什么是JSON

JSON(JavaScript Object Notation) 是一种轻量级的数据交换格式。它由javascript发展过来，但是任何语言都能对其进行解析。

与XML的联系区别

两者都是一种语言规范，JSON可以理解成为XML的简化版。但实质上与XML是无关的

区别在于：XML的可读性很高，但是其数据量大

JSON与之相反，虽然可读性比较差（比自定义的可读性还是强一点的），但是它是轻量级的，故数据量小，故我们进行数据交互多使用JSON

JSON定义方式：

键值对

{“firstName”:”Brett”}（此时不能体现价值，比键值对占的空间还大）

{} 对象
[] 数组
“” 双引号内是属性或值
: 冒号表示后者是前者的值

多个键值对

{“firstName”:”Brett”,”lastName”:”McLaughlin”,”email”:”aaaa”}

数组

多个数组

下面我们做一系列的判断题

{“sub_int”:”a”} true
{“sub_int”:a} false
{“sub_int”:100} true，对象
[{“sub_int”:100}] true，数组
{123:100} false，属性必须有双引号
{“123”:100} true
{“sub_int”:123.12} true
{“sub_int”:true} true
{“sub_int”:null} true
{‘sub_int’:’a’} false，不能用单引号
