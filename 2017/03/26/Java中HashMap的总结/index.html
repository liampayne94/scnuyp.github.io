Java中HashMap的总结

发表于 2017-03-26 | 阅读次数

定义

HashMap实现了Map接口，继承了AbstractMap,其中Map接口定义了键映射到值的方法（与arraylist类似）

构造函数

HashMap提供了三种构造函数的方法

这里涉及到两个概念初始容量和加载因子，他们是影响HashMap性能的关键因素。其中初始容量是HashMap中桶的数量，而初始容量表示HashMap创建时的容量

加载因子是指哈希表在在其容量自动增加之前可以达到多满的尺度，他衡量的是散列表的空间使用度，当负载因子越大，空间利用率越高，但是同时查找效率降低。反之会造成空间上的浪费，因此一般加载因子取0.75

要深入理解HashMap，必须了解他的数据结构

数据结构

HashMap采取数组加链表的存储方式来实现。亦即数组（散列桶）中的每一个元素都是链表，如下图：

从源码中可以看出，每次新建一个HashMap时，都会初始化一个table数组。table数组的元素为Entry节点。

其中Entry为HashMap的内部类，它包含了键key、值value、下一个节点next，以及hash值，这是非常重要的，正是由于Entry才构成了table数组的项为链表。
