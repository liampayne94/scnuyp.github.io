利用git使用coding.Net进行代码托管

coding.Net是一个代码托管和项目管理的平台，相当于国外的Github。使用了一段时间，感觉挺方便的，使用起来比Github还要简单点。

Git的下载和安装就且不介绍了，百度一下，windows、mac版的都能顺利的下载下来。下面简单介绍一下使用git把本地项目上传到coding.Net的一些命令行。

一、初次创建项目

1、创建本地仓库

在本地磁盘创建一个文件夹

mkdir scnuyp(目录名)——做为你代码的父文件夹。

git init——此时文件夹生成一个.git的隐藏文件夹

2、将本地修改上传到暂存区

git add . 或者 git add ——它会监控工作区的状态树，使用它会把工作时的所有变化提交到暂存区，包括文件内容修改(modified)以及新文件(new)，但不包括被删除的文件。

git commit –m ``"项目描述"“项目描述”——将索引内容添加到仓库中

3、提交到仓库

git remote add origin https:``// git.coding.net/用户名/项目名.git

git push origin master

此时上传成功！

二、初次从分支上传项目

1、新建一个分支

git branch xxx

2、切换到该分支

git checkout xxx

3、重复执行一、中的2,3条

此时上传成功！

三、服务端上面已经有代码，从分支上传

1、重复二中的1,2条，切换到该分支

2、从服务端获取该分支的最新版本

（此时有两种方法）

git pull origin <分值名> （相当于是从远程获取最新版本并merge到本地）

git fetch origin <分值名>（相当于是从远程获取最新版本到本地，不会自动merge）

相比之下，fetch方法会比较安全

3、重复执行一、中的2,3条

此时上传成功！
